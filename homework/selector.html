선택자(selector)

선택자 종류

  • 전체선택자 : *{} 반응속도가 느려지기 때문에 많이 사용하지는 않음.
  • 요소(type): tag{} html의 body영역 태그를 선택하는 선택자
  • id : #idname{}태그요소중 속성명이 id를 뜻하는 것 앞에 #기호를 붙이고, 해당하는 속성명을 이어서 붙여 선택
  • class: .class_name{} 태그 요소중 속성명이 class를 뜻하는 것 앞에 . 기호를 붙이고, 해당하는 속성명을 이어서 붙여 선택
  • 자손 선택자: #wrap tag{} 태그 / ID / class 등의 형태 뒤에 띄어쓰기를 사용하고 그 자손의 요소형태를 작성하여 선택하는 방법 (자식요소, 자손요소 모두 선택)
  • 자식 선택자: #wrap>tag{}태그(tag)/ID/class등의 형태 뒤에 >를 사용하고 그 자식(children)의 요소 형태를 작성하여 선택 (선택자의 작성이 매우 길어짐)
  • 가상(type) : ::before{content:""} / ::after{content:""} 요소의 앞(before), 또는 뒤(after)에 실제로 존재하지 않는 가상의 요소를 생성하여 사용하는 선택자 반드시 송성의 기능에 content:"";를 작성해야 한다. : 가상선택자는 기본 형태가 display:inline;
  • 상태 선택자: tag:hover{} / tag:active{} / tag:visited{} / checkbox | radio: checked{} / select:selected{} 태그 또는 요소의 기능과 밀접한 관계가 있으며, 그에 따른 상태의 기능을 판단하여 처리하고자 할때 사용하는 선택자
  • 속성 선택자: 태그(요소)의 속성을 기준으로 판단하는 선택자 속성을 뜻하는 기호는 []이며, 기호 내부에 속성을 그대로 입력하면 된다! code[href="aaa"]{} : code태그의 속성명이 href이며, 그 값이 aaa인 것을 선택 code[href^="aaa"]{} : a의 속성명 href이며, 값 시작이 'aaa'를 포함하여 시작하는 모든 것.(그래서 aaa도 포함) code[href$="aaa"]{} : a의 속성명 href이며, 값끝이 'com'으로 끝나는 것. code[href!="aaa"]{} : a의 속성명 href이며, 'aaa'가 아닌 다른 모든 것. (http제외) code[href*="aaa"]{} : a의 속성명 href이며, 값에 'aaa' 표시가 있는 모든 것. code[href|="aaa"]{} : a의 속성명 href이며, 'aaa'라는 단어가 있는 것들만 인식. (|는 단어만 인지한다.) naver /* id값이 test인 것을 선택하라!
  • 동위(형제)선택자:
    1. 인접형제 선택자'+': 동일한 레벨(형제)에서 바로 아래의 형제(동생)를 선택하는 것.
    2. 형제 선택자'-': 동일한 레벨(형제)에서 동생을 선택, 형제 선택자는 동일한 레벨에서만 선택이 가능하며, 먼저 작성된 선택자에서 후에 작성된 형제로 진행만 가능하다.
    3. nth선택자:
      1. li:nth-child(){}
        1. li:first-child
        2. li:last-child
      2. code:nth-of-type(){}
        1. code:first-of-type{}
        2. code:last-of-type{}

        nth: n번째에 기능을 넣으시오!
