font의 속성

font-style

글자의 기울기를 결정하는 속성.

  1. font-style:nomal; - 일반글꼴
  2. font-style:italic; - 기울어짐
  3. font-style:initial; - 필기체
  4. font-style:inherit; - 부모요소에 상속받는 글꼴

font-variant

영소문자를 작은 대문자로 나타내는 속성

  1. font-variant:nomal; - 일반글꼴
  2. font-variant:small-caps; - 영문 소문자를 크기가 작은 대문자로 변환.

font-weight

글꼴의 두께를 결정하는 속성

  1. font-weight:nomal; - 일반글꼴
  2. font-weight:bold; - 두꺼워짐

font-size

글꼴의 크기를 결정하는 속성

  1. font-size:nomal; - 일반글꼴
  2. font-size:20px;
  3. font-size:1em;
  4. font-size:10pt;

font-height

글꼴의 높이(간격)를 결정하는 속성

  1. font-height:100%; 각 줄의 글씨 간격이 붙어있음.
  2. font-height:1.em;

font-family

폰트의 가족을 결정지어주는 속성. font-family를 결정한느 속성값으로는 family-name과 generic-family이다.

  1. family name : 굴림/나눔고딕/verdana/dotum;
  2. generic-family : 글꼴의 종류 (ex)serif, sans-serif, monospace etc..

font shorthand

css에서 여러 폰트를 하나하나 지정하기 힘들 때, 한줄로 축약하는 방법

  • font:[font-style][font-variant][font-weight][font-family]
