Reset.css이란?

Reset이란 개념은, 모든 브라우저에서 통일된 화면을 볼 수 있도록 기본 값을 처음부터 초기화 시킨다는 의미.

쉽게 말해, IE/chrome/firefox/opera etc.. 다양한 브라우저 사용 시, margine이나 height, table 등의 효과가 서로 다르게 표현된다. 이러한 불편을 없애고, 동일한 결과를 보여주기 위해 값을 0으로 초기화 시키는 것을 Reset.css라고 한다.

작성 방법

body{margin:0px; padding:0px;}
