가상 선택자

이전 선택자(Before)

실제로 존재하지 않고, CSS효과로만 보이게 처리하는 것.

링크

이후 선택자(After)

Before와 같은 기능이며, 이후(뒤에)구현된다.

이후어딘가에 나타나게 하기.

가상 선택자의 종류

  • before
  • after
  • hover : 해당하는 곳에 마우스를 올렸을 때, 기능을 하는 것
  • linked : link의 기능을 하는 것
  • visited : 방문하는 것.
  • active : 마우스를 누른 채로 있는 상태.
  • secected : form에서 사용
  • checked : form에서 사용
  • root : 최초 상태로 세팅
