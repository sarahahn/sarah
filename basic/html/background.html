  2. background-image: 이미지 삽입(배경으로 처리됨) url("사용하려는 이미지 경로 / 이미지.확장자");
  3. background-repaet: 배경이미지 (background-image) 사용시 반복처리 여부를 확인
    • repaet
    • repaet-x
    • repaet-y
    • no-repaet
  4. background-position: 배경이미지 사용시 해당하는 이미지의 배치되는 위치를 설정 (x축, y축)
    • top/left/right/bottom/center : 위치를 직접적으로 설명하여 배치하는 방법
    • n%: x축과 y축의 위치를 %표기법으로 사용하여 배치 (정확한 위치를 구현하기 어렵다.)
    • 수치 : x축과 y축을 정확하게 수치로 입력하여 배치
    • - 값으로 사용하거나, 지정된 개체의 크기보다 클 경우 보이지 않을 수도 있다.
    • 정해진 크기와 다른 수치를 입력하여 배경이미지의 위치를 아이콘이나 다른 기능으로 처리하는 방법이 있다.(is)
    • IR(image replace), IS(image splite)
